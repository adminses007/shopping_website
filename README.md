# 🛒 购物网站

一个功能完整的购物网站系统，包含前端展示、购物车、订单管理和后台管理功能。

## ✨ 功能特性

### 🔹 前端功能
- **商品展示页面**：图片、价格、简介、库存显示
- **购物车功能**：添加/删除商品、数量调整、总价计算
- **用户系统**：注册、登录、退出
- **订单提交**：填写联系方式，提交订单

### 🔹 后端功能
- **用户管理**：用户注册、登录、权限控制
- **商品管理**：添加/修改商品信息
- **订单管理**：订单创建、状态跟踪
- **后台管理**：订单状态管理、商品管理

### 🔹 数据库设计
- **users 表**：用户名、密码、邮箱、电话、管理员权限
- **products 表**：商品名称、价格、库存、图片、描述
- **orders 表**：订单号、用户、总价、状态、联系方式
- **order_items 表**：订单项详情（商品、数量、价格）

## 🚀 快速开始

### 1. 安装依赖

```bash
cd shopping_website
pip install -r requirements.txt
```

### 2. 启动应用

```bash
python run.py
```

### 3. 访问网站

- **网站地址**：http://localhost:5000
- **管理员账户**：admin / admin123

## 📱 使用流程

### 用户购物流程
1. **浏览商品** → 在首页查看所有商品
2. **加入购物车** → 选择商品和数量，添加到购物车
3. **查看购物车** → 点击购物车图标查看已选商品
4. **提交订单** → 填写联系方式，提交订单
5. **等待处理** → 管理员会联系您确认订单

### 管理员操作流程
1. **登录管理后台** → 使用admin账户登录
2. **查看订单** → 在订单管理页面查看新订单
3. **联系客户** → 根据订单中的联系方式联系客户
4. **更新状态** → 确认收款后更新订单状态
5. **商品管理** → 添加新商品或修改现有商品

## 🛠️ 技术栈

- **后端**：Python Flask
- **数据库**：SQLite
- **前端**：HTML5 + CSS3 + JavaScript + Bootstrap 5
- **用户认证**：Flask-Login
- **文件上传**：Werkzeug

## 🔧 配置说明

### 数据库配置
- 默认使用SQLite数据库
- 数据库文件：`shopping_website.db`
- 首次运行会自动创建数据库和表结构

### 文件上传配置
- 上传目录：`static/uploads/`
- 支持格式：JPG、PNG、GIF
- 最大文件大小：16MB

### 安全配置
- 密码使用Werkzeug加密存储
- 会话管理使用Flask-Login
- 文件上传使用secure_filename安全处理

## 📊 订单状态说明

- **pending**：待处理（新订单）
- **processing**：处理中（已联系客户）
- **shipped**：已发货（已收款并发货）
- **completed**：已完成（交易完成）

## 🚀 部署建议

### 生产环境部署
1. 修改`config.py`中的`SECRET_KEY`
2. 使用MySQL或PostgreSQL替代SQLite
3. 配置Nginx反向代理
4. 使用Gunicorn作为WSGI服务器
5. 配置HTTPS证书

### 性能优化
- 启用数据库连接池
- 使用Redis缓存
- 图片CDN加速
- 数据库索引优化

# 🛒 购物网站项目说明

## 📋 项目概述

这是一个功能完整的购物网站系统，按照您的要求实现了以下功能：

### 🔹 1. 前端功能（HTML + CSS + JS）
- ✅ **商品展示页面**：图片、价格、简介、库存显示
- ✅ **购物车功能**：添加/删除商品、数量调整、总价计算
- ✅ **订单提交**：填写联系方式，提交订单到后端API
- ✅ **用户系统**：注册、登录、退出功能

### 🔹 2. 后端功能（Python Flask）
- ✅ **用户管理**：用户注册、登录、退出
- ✅ **商品管理**：管理员可以添加/修改商品
- ✅ **订单管理**：保存用户下单信息到数据库
- ✅ **后台管理界面**：查看所有订单，手动标记订单状态

### 🔹 3. 数据库设计（SQLite）
- ✅ **users 表**：用户名、密码、邮箱、电话
- ✅ **products 表**：商品名称、价格、库存、图片
- ✅ **orders 表**：订单号、用户、商品、数量、总价、状态
- ✅ **order_items 表**：订单项详情

## 🚀 购物流程

### 用户购物流程：
1. **浏览商品** → 在首页查看所有商品
2. **加入购物车** → 选择商品和数量，添加到购物车
3. **查看购物车** → 点击购物车图标查看已选商品
4. **提交订单** → 填写联系方式，提交订单
5. **等待处理** → 管理员会联系您确认订单

### 管理员操作流程：
1. **登录管理后台** → 使用admin账户登录
2. **查看订单** → 在订单管理页面查看新订单
3. **联系客户** → 根据订单中的联系方式联系客户
4. **更新状态** → 确认收款后更新订单状态
5. **商品管理** → 添加新商品或修改现有商品

## 📱 快速启动

### 方法二：手动启动
```bash

pip install -r requirements.txt

python run.py
```
